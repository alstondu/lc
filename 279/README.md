# 279. 完全平方数

[Leedcode 链接🔗](https://leetcode.cn/problems/perfect-squares/description/)  
[Code](https://github.com/alstondu/lc/blob/main/279/279.cpp)

### 题目

给你一个整数 ```n``` ，返回和为 ```n``` 的完全平方数的最少数量 。

完全平方数是一个整数，其值等于另一个整数的平方。

---

#### 示例：

#### 示例 1：

**输入**：```n = 12```

**输出**：```3``` 

**解释**：```12 = 4 + 4 + 4```

#### 示例 2：

**输入**：```n = 13```

**输出**：```2```

**解释**：```13 = 4 + 9```

---
